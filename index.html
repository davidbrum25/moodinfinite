Moodinfinite Help

Welcome!

Moodinfinite is a professional digital canvas for your ideas. Add images, text, arrows, connectors, checklist, measurements and more to create moodboards, brainstorm, or organize your thoughts with absolute freedom.

Core Concepts

  • Panning & Zooming: Use the mouse wheel to zoom and middle-click (or Space + left-click) to pan. On Touch, pinch to zoom and drag with two fingers to pan.
  • Selection: Click an item to select. Hold Shift to select multiple. Drag on empty space to create a selection box.
  • Context Menu: Right-click (or Long press on touch) anywhere to access canvas and item settings.
  • Connectors: Select an item to reveal blue connection ports. Drag from a port to another item to create a dynamic connector line.
  • Reroute Nodes: Double-click a connector line to add a reroute node, allowing you to shape the connection path.

Tools & Hotkeys

  • Select (A): Default tool for interacting with the canvas.
  • Add Image (I): Upload images from your device. You can also Paste (Ctrl+V) or Drag & Drop files directly.
  • Add Text (T): Create text boxes for titles and descriptions. Double-click to edit.
  • Add Comment (N): Create sticky-note style comments for annotations.
  • Add Link (K): Add hyperlinks to your board. Click the external link icon on the item or toolar to open the URL.
  • Add Text List (L): Create interactive checklists. Click checkboxes to toggle items and double-click to edit the list.
  • Draw (D): Freehand drawing and sketching.
  • Add Arrow (Shift+A): Draw directional arrows. Hold Shift to snap to 45° angles.
  • Add Box (B) & Circle (C): Create geometric shapes. Toggle between Fill and Stroke in the toolbar.
  • Measure (M): Draw a line to measure distances. Switch units (px, cm, in) from the selection toolbar.
  • Add Grid (Alt+G): Create customizable grids. Adjust rows and columns in the toolbar.
  • Eyedropper (E): Pick any color from the canvas to use as your current accent color.

Item Manipulation

When an item is selected, a context-aware toolbar appears with specific actions:

  • Layering: Use Home/End to move items to the very front/back. Use Page Up/Down to move one layer at a time.
  • Grouping (Ctrl+G): Combine items to move and transform them together. Ctrl+Shift+G to ungroup.
  • Group Ordered (Ctrl+Shift+O): Select multiple images and use this to neatly align and group them horizontally.
  • Transform: Press S for Scale or R for Rotate. Drag the handles to modify. Hold Shift to maintain aspect ratio or snap rotation.
  • Flipping: Press H to flip horizontally or V to flip vertically.
  • Pin (P): Lock an item to prevent accidental movement or changes.
  • Auto Align (Ctrl+Shift+A): Neatly arrange multiple selected items into a grid.
  • Delete (Del / Backspace): Remove selected items. Use Ctrl + Click to delete connectors or reroute nodes.

General Hotkeys

  • New Board: Shift+N
  • Open / Save Project: Ctrl+O / Ctrl+S
  • Export PNG: Shift+S (Download) or Shift+C (Copy to Clipboard)
  • Edit Operations: Ctrl+C (Copy), Ctrl+X (Cut), Ctrl+V (Paste), Ctrl+D (Duplicate)
  • Undo / Redo: Ctrl+Z / Ctrl+Shift+Z
  • Select All / Invert: Ctrl+A / Ctrl+I
  • Toggle Grid: G (Toggle visibility), Alt+G (Grid Tool)
  • Deselect / Cancel: Escape
